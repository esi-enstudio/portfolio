## Why Astro?

Astro is the web framework for content-driven websites. It ships zero JavaScript by default, ensuring your site loads instantly.

## Key Benefits

- **Zero JS by default**: Only ship JavaScript for interactive components
- **Islands architecture**: Use your favorite framework components
- **Content collections**: Type-safe content management
- **Image optimization**: Built-in image handling

## Getting Started

Start with `npm create astro@latest` and explore the amazing ecosystem.
